SNP Browser

SNP Browser explores the presence of genetic variation between five populations by the retrieval of Single Nucleotide Polymorphism (SNP) data, the most common type of genetic variation observed amongst humans. The SNP Browser primarily focuses on data for bi-allelic SNPs located across chromosome 22 for Gujarati, Gambian Mandinka, Japanese, Puerto Rican and Toscani populations, which is searchable by either rsID, HGNC gene names or aliases and genomic position. These search engines are connected to a comprehensive database that contains essential SNP data and observed allele and genotype frequencies across the five populations. A statistics tool is also available, which calculates selected statistics which uncover insights into population genetic variation, neutrality, and genetic and haplotype diversity.

Search Tools

rsID

Our database holds data on over 1 million SNPs located across chromosome 22, which are searchable by their appointed rsID, the standard naming convention for most SNPs.

Genes

The SNP Browser database contains data on the corresponding HGNC accepted gene names and aliases. The results will return all SNPs located within the region of the gene.

Genomic co-ordinates

SNP data can be retrieved by a set of genomic co-ordinates across chromosome 22. All SNPs found to be located within the specified region in our database will be returned.
